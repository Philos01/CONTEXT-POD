# 二、真正杀手级优化：引入“关系博弈状态机”

你现在 System Prompt 很强，但有个问题：

> ❌ 每次都是“独立决策”
> ✔️ 应该有“局势阶段判断”

---

## 🔧 加一层：Conversation State Engine

在调用 System Prompt 前，加一个轻量判断：

```ts
state = classifyState(rawText, history)
```

---

### 🧠 状态分类建议（核心）

```text
1. 试探期（对方在摸底）
2. 施压期（对方在要结果）
3. 冲突期（情绪/对抗）
4. 冷处理期（对方在降温/疏远）
5. 关系修复期
```

---

### 💡 然后你在 User Prompt 注入：

```text
当前关系态势判断：{state}
```

---

### 🔥 效果

同一句话：

> “这个事情怎么还没好？”

在不同 state 下，策略完全不同：

* 试探期 → 轻解释
* 施压期 → 给确定性
* 冲突期 → 先降温
* 冷处理 → 反拉关系

👉 这一步会让你的系统**像真人在“读空气”**
